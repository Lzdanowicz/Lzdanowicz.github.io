CSS Positioning

One of the most important things to note, when working with CSS is the fluidity of the elements within the webpage. There are a few different properties that elements can take form under when discussing this topic. A fixed positioning is an element that is fixed in size within the browser window, no matter how small the window is made the element will retain it's initial size. This form of positioning was popular in the days of a standard screen where developers did not have to worry how a webpage would form onto a smaller screen, but with the development of handset technology that notion has changed. Absolute positioning on the other hand states that an element will position itself relative to the first parent element that has a position other than static. This is a step beyond the fixed positioning since it does allot for fluidity between the elements. Finally there is Relative positioning in which and element is positioned reletive to its normal position in the flow of the site, this allows for added fluidity when adjusting to different screen and window sizes.
