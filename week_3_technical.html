Why Javascript is popular in Web-Development?

Before immersing myself in code at DevBootCamp, I wouldn't be able to tell you much about Javascript and its functions as a development language even though I would always hear of it's importance within the programming world. What seperates Javascript from many other languages is the fact that it is a dynamic language that allows room for interaction within objects in the environment. Because of this functionality Javascript is often used in the development of games as well as interactive web applications. Another interesting aspect of Javascript is the fact that it has allot of built in functions, something that I haven't seen much of in my brief expereince with programming languages. In addition Javascript code is also executed on the client's server which saves bandwith and puts less stress on the server.
